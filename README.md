## aws-route53-example

#### How to run:

1. Clone this repo;
2. Run `bundle install` inside;
3. Create config file `config.yml` like this:
  ```
  aws:
    key: 'YOUR-AWS-KEY-ID'
    secret: 'YOUR-AWS-SECRET'

  ```
4. Run the script `ruby route53.rb`.
